# M2 Functionalization Assay Examples

These files are templates for wet-lab `AssayRecord` JSONs. Keep real campaign
records in `../assays/`; the examples directory is intentionally not consumed
by the default CLI ingest path.

The records demonstrate ligand-density and activity-retention assay metadata
that DPSim can turn into `CalibrationEntry` objects with posterior uncertainty
and validity-domain information.
