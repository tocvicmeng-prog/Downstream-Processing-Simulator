# M3 Binding And Pressure-Flow Assay Examples

These files are templates for wet-lab `AssayRecord` JSONs. Keep real campaign
records in `../assays/`; the examples directory is intentionally not consumed
by the default CLI ingest path.

The examples cover the three P5/P5+ M3 calibration styles:

- Static binding isotherm points for weighted Langmuir fitting.
- Raw breakthrough curves for integrated DBC references.
- Pressure-flow curves for packed-bed hydraulic slope calibration.
